# NETFILX_user_service
ASEE Project, content service part.

## ENTITY CLASSES
### User class
- nome: String
- surname: String
- password: String
- email: String
- dateOfBirthday: Date
- paymentMethod: String
- isActive: Boolean

### User service
- /users
    - GET
    - POST
- /users/{userId}
    - GET
    - PUT
    - DELETE
- /users/{userId}/profile
    - GET
    - POST
- /users/{userId}/profile/profile{id}
    - GET
    - PUT
    - DELETE
